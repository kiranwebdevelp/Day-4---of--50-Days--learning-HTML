HTML Attributes

About This Page

This page demonstrates various HTML attributes such as class, id, style, and media attributes like src and alt.

Common HTML Attributes

class: Assigns one or more classes to an element, which can be used for styling or JavaScript manipulation. <p class="highlight">This paragraph has a highlight class.</p>
id: Assigns a unique identifier to an element, which can be used for styling, JavaScript manipulation, or linking. <p id="intro">This paragraph has an ID of "intro".</p>
style: Adds inline styles to an element, overriding any external styles. <p style="color: red; font-size: 24px;">This paragraph has inline styles.</p>
title: Provides a tooltip or hover text for an element. <p title="This is a tooltip">Hover over me!</p>
alt: Specifies alternative text for an image, which is displayed if the image cannot be loaded. <img src="image.jpg" alt="A beautiful mountain scenery">
src: Specifies the source URL of an image, script, or link. <img src="image.jpg">
href: Specifies the link URL for an anchor element. <a href="https://www.example.com">Visit example.com</a>
target: Specifies the target window or frame for a link. <a href="https://www.example.com" target="_blank">Open in new tab</a>
rel: Specifies the relationship between the current document and the linked document. <a href="https://www.example.com" rel="nofollow">No follow link</a>
data-*: Custom data attributes, which can be used to store additional information about an element. <p data-author="John Doe">This paragraph has a custom data attribute.</p>

In-Depth Concept: The class Attribute

The class attribute is used to assign one or more classes to an HTML element. Classes are used to group elements together, making it easier to style or manipulate them using CSS or JavaScript.

Multiple Classes

An element can have multiple classes, separated by spaces.

<p class="highlight important">This paragraph has two classes.</p>
